Tired of that annoying chat?

Why not use it the way a chat SCREEN is ment to be used. Like a popup.
Just install Greasemonkey for firefox or tampermonkey for chrome and install this plugin.
Than you are ready to go and rock your chat screen on every Grepolis server!
